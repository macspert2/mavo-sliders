This is a project for a wordpress plugin which visually shows the same 2 sliders to the visitor that are visible on http://www.mamnvoyage.com/
These sliders are currently generated by the Smart Slider plugin.
The goal of the plugin is to generate visually identical output in a much slimmer package. The Smart Slider plugin has a lot of unused functionality.
Specifically:
top left slider (in the sticky bar) - slider #3:
- clicking anywhere on this slider always leads to the homepage, no matter which slide it is on
- 2 slides alternate: an image (the logo) and a text slide (blog title and subtitle)
- on the English (en) and German (de) version of the site, the subtitle is in the respective language and the link points back to the respective language home page
the hero slider (just below the sticky bar) - slider #2:
- has 5 slides (down from currently 6)
- first slide is again the logo of the blog in full width behing the blog title (the blog title has a content-wide transparent blue backgroound to visually seperate it from the image behind it, clicking anywhere leads to the homepage
- slides 2-5 show a randomly selected blog post each, the post's featured image in full width behind the post title (title has the same transparent blue background as on slide 1°, clicking anywhere on this slide leads to the respective post

Be proactive and inventive - slim down html and css where possible without impacting visual output, add optimisations for mobile (srcset for images) where this can help with performance
